Hyper Text Markup Language (HTML)

HTML (Hyper Text Markup Language) is a programming language that communicates with the web browser to display content on to the web browser.

HTML use elements to document on how the web page would looks like. The elements can be altered by attributes which modify how the element will be displayed on to the web page.

What are elements?

Elements document how the web page will be displayed.

They begin with a less than symbol and end with a greater than symbol

What are attributes?

Attributes modify the elements to be display differently.

They can increase an image height, width or the color of a text

The Basic Elements

These elements are used in almost every web page that you can think of.

The basic skeleton of HTML is made of four elements called HTML, Heading, Title, and Body.

There are two others called the P element and H1 to H6 element which is used to create context for the web page .

HTML Element

It sets the tone for the web browser and notify the web browser that HTML code is in this file.

All other elements must be between the HTML element to be displayed properly.

Body Element

In the body element, it will contain all your work and code.

It will display all the context on to the web page

Heading Element

The core element of HTML and it gives a description of the web page

Contains the Title element which displays the web page title.

Title Element

This element goes inside the Heading element.

It gives the web page a title so users can understand the name of the web page.

H1 to H6 Element

These elements are large fonts with the bold style for a web page heading.

H1 is the largest font. While increasing toward to H6, H6 is the smallest heading font

They are normally used for titles for web articles or to grab users attention.

P Element

The P element also known as the paragraph element creates a paragraph.

This is normally used to create web articles or display large amount of text

Adding Images

This element will take a image from a directory and display the image.

You can use attributes to modfiy the image. Height and width can change the image size displayed on the web page but does not directly alter the actual file.

Types of image files that HTML can read are PNG, GIF, and JPEG. There are other files but these are the most common image files.

Code: <img src="File.png">

Links/Anchor Element

This element is used to link one html file to the next html file.

It can also be used to link to another website as well.

The href attribute specifics the URL link that you want to connect to when the user clicks on the link.

Href stands for Hypertext Referebce.

Code: <a href="link.html">Link</a>

<a href="http://www.google.com">Google</a>
