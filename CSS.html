Cascade Style Sheet (CSS)

CSS (Cascade Style Sheet) is a programming language that communicates with the web browser on how HTML elements should be displayed.

HTML elements use a default setting on how it is displayed on the web browser however CSS allows greater manipulation of these elements to be displayed even more beautifully and more control for designers

Saving your CSS file and connecting it to your HTML File

Just as it is important to save your HTML with a HTML extention at the end of the file name, it applies to CSS files. Add .css extention on the file name

CSS allows you to connect to multiple HTML file using only one CSS file therefore allowing you to make quick easy changes

CSS Selectors

Selectors select elements from the HTML that you wish to change or alternate.

There are three types of selectors. Element selector, Class selector, and ID selector.

Element Selector

Element selectors targets a element name and applies the changes to all the elements with the same name.

An example would be a p element with CSS code that changes the font to 22 pixel.

This causes all p elements in the HTML file to have a font of 22 pixel. So keep this in mind that this will affect the same element throughout the web page.

Class Selector

The class selector targets specific elements with a specific name.

This is very useful if you want to apply a certain CSS code for a specific part of the web page.

The class selector can be used multiple times. So if you need to apply a CSS code with redundancy. Class selector will make it easy.

ID Selector

The ID selector is similar to the class selector. It targets a specific name of an element.

However this can only be used once unlike the class selector.

Use this if you think you only need to use the bracket of CSS code only once.

Color

The property name for color is color.

The color property is mainly used for texts in the web page.

The value of color could be: RED, YELLOW, GREEN, BLUE, BLACK, WHITE, BROWN, and etc.

You can use hexadecimal as a value to get different shades of a specific color.

Background-Color

The property name for background color is background-color

Background color is normally used to color the background of the web page.

The value for background color could be: RED, YELLOW, GREEN, BLUE, BLACK, WHITE, BROWN, and etc.

You can use hexadecimal as a value to get different shades of a specific color.

Font Family

The property name for the font family is font-family.

The font family property is mainly used for texts in the web page. So if you want to make the font artistic you can.

The value of font-family could be: verdana, serif, sans-serif, monospace, and etc.

There are more fonts that you can get from online.

Text-alignment

The property name for text-alignment color is text-align

This is normally used to change the alignment of a text or paragraph somewhere in the web page.

The value for text-align could be: left, right, center, or justify

Font Size

The property name for the font size is font-size.

The font size property is mainly used for texts in the web page. So if you want to change how large the text is, you can.

The value of font-size could be: any pixel number. Example would be 32px.
