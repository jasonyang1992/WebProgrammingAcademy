JavaScript (JS)

JavaScript is a programming language that determines the behavior of the web browser.

An example would be data validation. Prompting users to enter only numbers for a zip code whenever they enter letters.

Internal JS vs External JS

If you plan to repeat JS code on to other HTML files it is better to use external JS because it can cascade to other HTML pages

It also useful to keep all your JS code into one area if you plan to make future changes to make things easier

Internal JS may get messy because it can get bury by all the HTML code making it very difficult to keep track where the code is.

Functions

Functions can be a list of commands or code that you want to initiate when a condition happens.

An example will be if the user press a button three times, it will say you have press this button three times.

Functions can also be recycle over and over if a certain code needs to be used again.

Variables

Variables in javascript are similar to algebra. They hold a value to be used in conjunction with operators.

Examples like creating a counter for number of users clicking on a button. A variable can hold that information and then written in text.

Variables can be used in many ways as long as you are creative on how they can be used.

Code: var x = 3; var y = 2;

Operators

Operators are additon, subtraction, multiplication and division. They work in conjunction with variables

You can make a simple calculator using operatiors with variables

You can also create a counter that by adding one or subtracting one.
